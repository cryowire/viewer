<h1 align="center">cryowire/viewer</h1>
<p align="center">Read-only viewer for <code>cooldown.yaml</code> files generated by <a href="https://github.com/cryowire/core">cryowire</a>.</p>
<p align="center">
  <a href="https://cryowire.github.io/"><img src="https://img.shields.io/badge/Website-cryowire.github.io-38bdf8?style=for-the-badge" alt="Website" /></a>
  <a href="https://cryowire.github.io/viewer/"><img src="https://img.shields.io/badge/Live-Demo-22c55e?style=for-the-badge" alt="Live Demo" /></a>
</p>

Drop a `cooldown.yaml` file onto the page to view:
- Cooldown metadata (ID, date, cryo, operator, notes)
- Wiring summary tables (attenuation, gain, per-stage components)
- Line detail browser with component information

## Live

<https://cryowire.github.io/viewer/>

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Deploys to GitHub Pages automatically on push to `main`.
