# cryo-wiring viewer

Read-only viewer for `cooldown.yaml` files generated by [cryo-wiring-core](https://github.com/cryo-wiring/core).

Drop a `cooldown.yaml` file onto the page to view:
- Cooldown metadata (ID, date, fridge, operator, notes)
- Wiring summary tables (attenuation, gain, per-stage components)
- Line detail browser with component information

## Live

<https://cryo-wiring.github.io/viewer/>

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Deploys to GitHub Pages automatically on push to `main`.
